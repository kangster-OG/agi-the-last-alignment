## Replacement Rules

- Replacing a V1 file with licensed or commissioned material requires updating this file and `src/audio/audio_manifest.json`.
- BOOM Library, A Sound Effect, Sonniss, Kenney CC0, Freesound CC0/CC-BY, or OpenGameArt CC0/CC-BY can be used only after saving the exact source URL/license/author and confirming redistribution inside the browser game is allowed.
- Adobe Stock audio requires an extended game/app license and should not be used under a standard audio license.
- Suno, ElevenLabs, Udio, or similar generated audio stays reference-only unless the user explicitly approves production use after checking current terms.
