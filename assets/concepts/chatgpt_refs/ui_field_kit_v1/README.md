## Prompt Intent

Create a bright, modular, high-fidelity pixel-art UI kit that combines:

- Civic Accord field hardware: pale ceramic plates, repaired civic equipment, treaty amber markers, teal terminal glass.
- Overworld Signal route language: node tokens, route contract boards, signal strips, modular tabs, cable sockets.

This source was used for the first runtime field-kit attempt. The user rejected that direction as not matching the accepted Armistice/global art style closely enough, especially for HUD, dialogue, pickup callouts, draft/item boards, and camp/summary panels.

## Direction Notes

- Overall value should stay mid-bright and readable across future levels, not locked to a dark Armistice-only mood.
- Panels should feel physical and manufactured: chipped enamel, light metal, terminal glass, tiny bolts, colored signal tabs.
- Violet corruption is an accent only.
- Avoid flat web cards, purple gradients, black-heavy boxes, and generic sci-fi dashboard panels.
