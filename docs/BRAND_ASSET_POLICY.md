# Brand Asset Policy

This project is intended to be a free, MIT-licensed, open-source parody game.

The user wants real frontier AI lab names and official logos to appear as part of the parody/faction language.

## Creative Direction

- Official logos may be used when they clearly serve the parody/faction presentation.
- Real lab/company names may be used as fictional faction names.
- The game should remain obviously unaffiliated, fictional, satirical, and transformative.
- The tone should be broad, absurd, and mythic rather than making literal claims about real organizations.

## Implementation Rules

- Keep official logos in a clearly separated third-party asset area, such as `assets/third_party/logos/`.
- Record each logo in `ART_PROVENANCE.md`.
- Do not list official logos as original project-created art.
- Do not mark official logos as MIT-owned original assets unless the asset license independently allows it.
- Add a visible repository/menu/credits disclaimer:

> This is an unaffiliated fictional parody project. Real company and model names/logos may be used for satirical worldbuilding. No endorsement by any referenced organization is implied.

## Still Avoid

Unless the user explicitly directs otherwise, avoid copying:

- official product UI screens
- proprietary app interfaces
- mascots or character art
- marketing layouts
- ad copy
- screenshots of real services

Use logos as faction identifiers, but build the game UI, portraits, units, animations, maps, and effects as original work.
